Theme Options

As a free addition to this theme, we have included a unique Chatloop-chat. Once activated, it allows expanding the options of the theme noticeably. These Chatloop option allow you to use custom redux option.
After plug-in's installation, the "Chatloop Option" menu item should appear in the "Theme Panel" tab in WordPress dashboard menu panel.

General Settings

This option are responsible for managing site overall look. In this option you should set Theme Stylesheet, change theme layout using Rtl mode. You can also set preloder visibality,custom css code,custom js code and also set back to top button visibality.



Header

This option are responsible for managing site header design . In this option you should set transparent header, sticky header, sticky nav background color,nav item color,nav item hover color.

Footer

This option are responsible for managing site footer page content. In this option you can add footer column,footer copyright text,footer background color,copyright background color,social media icon and links. In this option you can change visibality of footer social media area and rediract that social media link option setting.Also you can change footer copyright area in this application.



Page

This option are responsible for managing site page layout. In this option you can set page layout position.

Post

Bolg & Archive Page

This option are responsible for managing site Blog page layout. In this option you can change blog & archive page layout position,blog column parts,reade more option and also provide blog & archive page author,blog comment count detail and read more link option display or not(Enable/Disable).

Single Post

This option are responsible for managing site Single post layout. In this option you can set single post layout positiona and also provide blog & archive pageauthor name and blog comment count detail display or not(Enable/Disable).

Color Option

This option are responsible for managing site post page layout. In this option you can change post title color,post text color,post meta data color.

Sidebar

This option are responsible for managing site sidebar content . In this option you should change sidebar title color,sidebar text color also set sticky sidebar(position).

Search Page

This option are responsible for managing site Search page layout. In this option you can set search page layout position.

404 Error Page

This option are responsible for managing site Search page layout. In this option you can set search page layout position.

Typography

This option are responsible for managing site conatnt design. In this option you should change theme Background color ,conatnt font style,size and color.
