Custom Page Option


A custom page type is an additional type of page that is tailored for a specific kind of content. Custom page types are a part of the Chatloop-chat plugin, and you can use them after the plugin activation.

Banner Settings

In this option you can change main page banner color and change breadcrumb position using this custom page option.



In this option provide "banner setting option" when you can click on that button at that time that button posstion will be change (Enable/Disable). Banner Visiblity is totally depands on this button value. so, you can easely change this banner setting based on that button value(Enable/Disable).



In this option you can change breadcrumb position using this custom page option.you can set the position of the breadcrumb like this:

  1. Right Side
  2. Center
  3. Left Side

Menubar Option

In this option you can change manubar position using this custom page option.you can set the position of the manu like this:
